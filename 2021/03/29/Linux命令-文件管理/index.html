1.查看文件信息：ls

ls是英文单词list的简写，其功能为列出目录的内容，是用户最常用的命令之一，它类似于DOS下的dir命令。

Linux文件或者目录名称最长可以有265个字符，“.”代表当前目录，“..”代表上一级目录，以“.”开头的文件为隐藏文件，需要用 -a 参数才能显示。

语法说明：

1. ls 显示当前目录里面所有文件及文件夹

2. ls -a 显示当前目录下所有文件包含隐藏文件

3. ls -al 显示当前目录下的所有文件包含隐藏文件，以列表方式显示文件的详细信息

4. ls -alh显示当前文件下的所有文件包含隐藏文件，以列表方式显示文件详细信息和大小

5.ls -dl查看当前文件夹的属性

图中的信息：

  1. d代表文件类型，rwx代表文件存取控制
  2. 14代表硬连接数
  3. 第一个admin代表文件属主，第二个admin代表文件所属的组
  4. 4096代表文件大小
  5. 3月后面代表文件修改时间
  6. 最后一个参数是文件名

与DOS下的文件操作类似，在Unix/Linux系统中，也同样允许使用特殊字符来同时引用多个文件名，这些特殊字符被称为通配符。

15.建立连接文件：ln

Linux链接文件类似于Windows下的快捷方式。

链接文件分为软链接和硬链接。

软连接：软连接不占用磁盘空间，源文件删除则软连接失效。

硬连接：硬链接只能连接普通文件，不能连接目录，会同步更新。

格式：

​ ln 源文件 链接文件

​ ln -s 源文件 链接文件

如果没有-s选项代表建立一个硬链接文件，两个文件占用相同大小的硬盘空间，即使删除了源文件，链接文件还是存在，所以-s选项是常见的形式。

注意：如果软链接文件和源文件不在同一个目录，源文件要使用绝对路径，不能使用相对路径。

16.输出重定向命令

Linux允许将命令执行结果重定向到一个文件，本应显示在终端的内容保存到指定文件中。

如：ls>test.txt(test.txt如果不存在，则创建，存在则覆盖其内容)

cat -n test.txt查询看可以显示行号

总结：所谓的重定向就是修改了默认的输出方向

注意： >输出重定向会覆盖原来的内容，>>输出重定向则会追加到文件的尾部。

19.文件搜索命令：grep

搜索文件里面内容的命令

Linux系统中grep命令是一种强大的文本搜索工具，grep允许对文本文件进行模式查找。如果找到匹配模式， grep打印包含模式的所有行。

grep一般格式为：

grep [-选项] ‘搜索内容串’文件名

在grep命令中输入字符串参数时，最好引号或双引号括起来。例如： grep’a’1.txt。

常用选项：

grep搜索内容串可以是正则表达式。

正则表达式是对字符串操作的一种逻辑公式，就是用事先定义好的一些特定字符、及这些特定字符的组合，组成一个“规则字符串”，这个“规则字符串”用来表达对字符串的一种过滤逻辑。

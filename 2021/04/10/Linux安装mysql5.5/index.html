Linux安装mysql5.5

发表于 2021-04-10 更新于 2021-04-11 阅读次数：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

1.下载

2.解压到/usr/local/mysql 【必须是这个目录】

3.安装数据库

  1. 进入安装mysql软件目录：执行命令cd/usr/local/mysql/
  2. 创建mysql用户: 执行命令useradd mysql
  3. 修改当前目录拥有者为mysql用户:执行命令chown -R mysql:mysql ./ [如果是就不用修改了]
  4. 安装数据库: 执行命令 ./sripts/mysql_install_db
  5. 把mysql目录所有者变成root: 执行命令chown -R root:root ./
  6. 到此数据库安装完毕

4.启动mysql服务和添加开机启动mysql服务

添加开机启动: 执行命令cp support-files/mysql.server /etc/init.d/mysql,把启动脚本放到开机初始化目录

启动mysql服务: 执行命令service mysql start

执行命令：ps -ef|grep mysql 看到mysql服务说明启动成功

5.修改mysql的root用户密码，root初始密码为空的

执行命令：./bin/mysqladmin -u root password ‘密码’

如果出现上面的问题

找到mysql.sock文件。使用一个软连接到tmp/mysql.sock

6.把mysql客户端放到默认路径

ln -s /usr/local/mysql/bin/mysql /usr/local/bin/mysql

注意：建议使用软链过去，不要直接包文件复制，便于系统安装多个版本的mysql

7.登录mysql -uroot -p

回车输入上面设置的密码

8.设置数据库默认编码

查看数据库编码show variables like ‘character_set_%’

找到系统里面的/etc/my.cnf文件

修改my.cnf文件

在如下位置上面添加

character_set_server=utf8

再次查询看编码

进入linux的mysql

执行use mysql

11，yum安装

1. 查看服务器中有没有安装过MySQL

  1. 查看是否还有mysql软件:

    rpm -qa|grep mysql

  2. 安装mysql客户端

    yum install mysql-server

    yum install mysql-devel

2. 启动&&停止 数据库字符集设置

  1. 配置mysql文件:

    ​ > cd /etc/my.cnf

    # 加入配置参数

    > character-set-server=utf8

  2. 启动mysql服务

    > service mysqld start

    #或者下面这个

    >/etc/init.d/mysqld start

  3. 设置开机启动

    >chkconfig –add mysqld

    >chkconfig mysqld on

  4. 查看开机启动设置是否成功

    >chkconfig –list | grep mysql*

    # mysqld 0:关闭 1:关闭 2:启用 3:启用 4:启用 5:启用 6:关闭停止

  5. 修改密码

    mysqladmin -u root -p password ‘123456’

3. 登录 mysql

>mysql -u root -p

输入密码

5. Linux MySQL 几个重要 的目录

数据库目录 /var/lib/mysql/

配置文件 /usr/share /mysql（mysql.server命令及配置文件）

相关命令 /usr/bin（mysqladmin mysqldump等命令）

启动脚本 /etc/rc.d/init.d/（启动脚本文件mysql的目录）

6.删除mysql数据库

#如果使用的是yum安装的mysql,需要删除的话,就是用如下命令:

> yum -y remove mysql*

# 然后将/var/lib/mysql文件夹下的所有文件都删除干净

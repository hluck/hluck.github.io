1.安装和配置jdk

JDK和JRE

JDK

1.JDK的全称（Java Development Kit）Java开发工具包

2.JDK = JRE + Java的开发工具（java,javac,javadoc等）

JRE

1.JRE(Java Runtime Environment) Java运行环境

2.JRE = JVM + Java的核心类库

3.想要运行一个开发好的Java程序，计算机只需安装JRE即可。

安装步骤

1.傻瓜式安装，下一步即可。

2.安装路径不要有中文或者特殊符号如空格等。

配置环境

1.我的电脑–属性–高级系统设置–环境变量

2.增加 JAVA_HOME 环境变量

3.编辑path环境变量，增加%JAVA_HOME%\bin

4.打开dos命令，任意目录下敲入java,如果出现javac的参数信息，配置成功。

2.相对路径和绝对路径

相对路径：从当前目录开始定位，形成的一个路径

绝对定位：从顶级目录（根目录）开始定位，形成的路径

需求：从d:\abc\test100访问d:\abc2\test200\hello.txt

3.Java常用转义字符

  1. \t : 一个制表位，实现对齐功能
  2. \n : 换行符
  3. \ \ :一个 \
  4. \ “ : 一个”
  5. \ ‘ :一个’
  6. \ r : 一个回车 System.out.println(“学习\r入门知识”);

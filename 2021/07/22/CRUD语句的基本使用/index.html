发表于 2021-07-22 更新于 2021-07-28 阅读次数：
本文字数： 3k 阅读时长 ≈ 3 分钟

1.Insert语句

语法：

*INSERT INTO table_name [ (column [,column1…]) ] *

VALUES (value [,value…]);

注意：

  1. 列可以插入空值，前提是该字段允许为空，insert into table value(null)
  2. insert into table(列名…) values(…),(…),(…)形式插入多条数据
  3. 如果是给表中的所有字段添加数据，可以不写前面的的字段名称
  4. 默认值的使用，当不给某个字段值时，如果有默认值就会添加，否则报错

2.Update语句

UPDATE table_name

​ SET col_name1 = expr1 [,col_name2 = expr2 …]

​ [WHERE where_definition]

3.Delete语句

delete from table_name

​ [WHERE where_definition]

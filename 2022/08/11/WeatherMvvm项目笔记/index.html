Android网络安全配置

​ 在 Android 7.0了，引入了「Network Security Configuration」，当时默认配置为信任系统内置证书以及用户安装证书，并且允许访问 HTTP 请求。

​ 而在 Android 9.0上，默认只信任系统证书了，并且默认不再允许访问 HTTP 请求。

详情请看文章：网络安全之NetworkSecurityConfig

官方网络安全说明：https://developer.android.google.cn/training/articles/security-config

官方网络安全教程:https://developer.android.google.cn/codelabs/android-network-security-config#0

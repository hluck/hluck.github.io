Ps提升课 抠图二

发表于 2020-11-10 阅读次数：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

1.简单抠图

1.使用工具： 选框工具、多边形套索工具、钢笔工具

适用范围：边界清晰流畅，图像内部也没有透明的对象，其外形为基本的几何型

2.使用工具： 魔棒、魔术橡皮擦、背景橡皮擦、快速选择工具、磁性套索工具

适用范围：对象与背景之间有较强的色彩差异。

2.需要用到的工具

证件照————色彩范围、调整图层、蒙版

抠头发（简单背景）————钢笔、通道、色阶、图层蒙版、画笔、混合模式

抠头发（复杂背景）————钢笔、通道、色阶、图层蒙版、画笔、混合模式、分区调整

抠毛发————选择并遮住

3.抠图方法

1.证件照(单位是cm,分辨率是300）————色彩范围

  1. 新建2.5*3.5cm的一寸照片画布，置入图片，菜单栏-选择-色彩范围（alt+s+c),吸管点击背景，背景选择不全面可调整容差或者使用加选小吸管。
  2. 点击确定，选择的是背景区域，ctrl+shif+i反选选择人物，ctrl+j复制。
  3. 新建图层，调整图层顺序到抠好的图层下方，填充颜色。
  4. 为抠好的图层添加色相/饱和度，选择黄色区域，降低饱和度。将色相/饱和度添加剪贴蒙版到抠好的人物图层上。
  5. 可以新建空白图层，剪贴到抠好的人物图层上，利用画笔工具吸取头发颜色降低不透明度涂抹在头发边缘遮住原本环境色。
  6. 根据背景颜色调整色彩平衡，统一画面色调。
  7. ctrl shift alt e盖印图层。
  8. 新建8.9*12.7cm的5寸画布，把盖印好的寸照拿到5寸画布进行排版即可。

2.抠头发（简单背景）——通道

  1. 钢笔工具将人物和背景分离，路径转选区ctrl+enter，ctrl j复制。
  2. 观察RGB通道，选取对比强烈的通道进行拷贝。
  3. 执行ctlr L调出色阶，增强对比。因为头发是深色，所以背景需要调整成白色最终才能抠干净。
  4. 按住ctrl点击通道缩览图，载入选区，ctrl+shift+i反选（注意通道载入选区默认为白色，但是我们需要提取的头发颜色是黑色）。
  5. 为下方原人物图层添加图层蒙版，擦除多余背景部分，保留人物。
  6. 新建空白图层，剪贴到抠好的头发图层上，利用画笔工具吸取头发颜色降低不透明度涂抹在头发边缘遮住原本环境色。
  7. 根据画面情况适当调整颜色。

3.抠头发（复杂背景）——通道

  1. 钢笔工具将人物和背景分离，路径转选区ctrl+enter，ctrl j复制。
  2. 观察通道，选择一个对比最舒适边缘最干净的通道进行拷贝，框选出一部分背景颜色相似的区域，给选区进行羽化，使用色阶进行调整或者使用白场吸管直接点击背景颜色变白。
  3. 再使用套锁工具框分别选中剩余的头发和背景区域，给选区进行羽化，使用色阶进行调整或者使用白场吸管直接点击背景颜色变白。
  4. 按住ctrl点击通道缩览图，回到rgb复合通道，回到图层，ctrl+j复制。
  5. 给下方图层添加蒙版，擦拭多余边缘。

4.抠毛发（动物）——选择并遮住

  1. 使用快速选择大概框选出狗子边缘，ctrl +j复制。
  2. 选择并遮住/调整边缘（所有能形成选区的工具属性栏都有），根据图片情况选择视图模式。
  3. 边缘检测中勾选智能半径（可自动缩进边缘），适当调整全局调整中的滑块，输出设置选择净化颜色，输出到新建带有图层蒙版的图层。
  4. 使用调整边缘画笔在四周擦拭掉多余背景，确定即可。
